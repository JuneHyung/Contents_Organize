## Node. JS와 NPM

### [NPM](https://www.npmjs.com/)

**Node Package Manager**

자바스크립트 프로그래밍 언어를 위한 패키지 관리자

```shell
노드 버전 확인 : node -v
npm 버전 확인 : npm -v
```



### NPM 초기화 명령어 - init

```shell
npm -init
```

* package name : 
* version
* description
* entry point
* test command
* git repository
* keyword
* author
* license
* ...

최종적으로 나오는 모습을 보여주고 OK

package.json이 생성된다.
