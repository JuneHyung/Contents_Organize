# Event

[Event](https://apexcharts.com/docs/events/)

해당 이벤트가 트리거 될때 고유한 기능을 연결할 수있는 이벤트

## 사용가능 이벤트

* animationEnd
* beforeMount
* mounted
* updated
* mouseMove
* mouseLeave
* click
* legendClick
* markerClick
* selection
* dataPointSelection
* dataPointMouseEnter
* dataPointMouseLeave
* beforeZoom
* beforeResetZoom
* zommed
* scrolled
* brushScrolled



옵션의 chart.events에서 자세한 내용 참고.

[chart.events](https://apexcharts.com/docs/options/chart/events/)
