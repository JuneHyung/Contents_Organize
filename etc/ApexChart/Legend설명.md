# Legend

차트에 여러 dataSeries가 있는 경우 범례는 미리 정의된 기호와 계열 이름으로 각 dataSeries를 식별하는데 도움



## 표시 / 숨기기

```javascript
 legend: {
    show: false
  }
```



## Position

* top
* right
* bototm
* left

```javascript
legend: {
  position: 'top'
}
```



## Alignment

* cemter
* left
* right

```javascript
legend: {
  horizontalAlign: 'right',
}
```
